# Day 2: Patient Agent (Chatbot & Appointments)

Phase 2 focused on empowering patients with an intelligent, 24/7 personal health companion.

## 👤 Patient Dashboard
- **Medication Tracker**: Real-time view of active prescriptions with dosage and timing.
- **Health Snapshot**: Integration of vitals tracking and upcoming appointment reminders.

## 💬 AI Health Companion
- **WhatsApp-Style Interface**: A familiar, conversational UI for patients to interact with the AI.
- **Symptom Triage**: The agent analyzes patient-reported symptoms and provides priority-based guidance.
- **Context-Aware Answers**: The AI reads the patient's record to answer specific questions like "When is my next dose?".

## 📅 Smart Appointment Booking
- **Specialty Matching**: Patients can select specialists (Cardiology, General, etc.) and view available doctors.
- **Calendar Integration**: A 3-step scheduling wizard that writes directly to the hospital database.
